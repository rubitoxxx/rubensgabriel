## 📂 Estrutura de Páginas

| Página | Descrição |
|--------|-----------|
| `index.html` | Página inicial com menu de navegação para os projetos |
| `python.html` | Página sobre Python (inclui imagem `python.png`) |
| `java.html` | Página sobre Java (com imagem `java.png`) |
| `webdesign.html` | Conteúdo introdutório sobre Web Design |
| `calc.html` | Calculadora funcional em JavaScript |

| `musica.html` | Página de exemplo relacionada à música |
| `rubix.html` | Página que possivelmente redireciona ao projeto [Mural Rubix](https://rubitoxxx.github.io/rubix/) |
